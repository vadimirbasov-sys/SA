1. Places (Места)
    GET /places  Получить список мест с фильтрами.
    GET /places/{id}  Получить детали места по ID.
    GET /places/search  Поиск мест по запросу.
    GET /places/{id}/reviews  Получить отзывы места.
    POST /places/{id}/reviews  Добавить отзыв к месту (требует аутентификации).

2. Categories (Категории)
    GET /categories  Получить список категорий с количеством мест.





3. User (Пользователь)
    POST /auth/login  Вход пользователя.
    POST /auth/register  Регистрация нового пользователя.
    GET /user/favorites  Получить избранные места пользователя (требует аутентификации).
    POST /user/favorites  Добавить место в избранное (требует аутентификации).
    DELETE /user/favorites/{placeId}  Удалить место из избранного (требует аутентификации).

4. Working Mode (Режим работы)
    GET /working-mode  Получить информацию о режиме работы (глобальном или для места).

5. Дополнительные (Карта и кластеры)
    GET /places/clusters  Получить кластеры мест для карты.
